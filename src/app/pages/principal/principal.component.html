¿Qué aprenderás?

Desde conocimiento básicos hasta avanzados para el mundo de la programación

Back End

Para el back end utilizaremos uno de los lenguajes más populares y con mas tiempo en el mercado.

Base de Datos

Para el almacenamiento de los datos utilizaremos un motor rápido, confiable y gratuito.

Front End

Aprenderás lo básico de HTML, CSS y Javascript haciendo hincapié en este último por la popularidad en los últimos años.

Repositorio

Utilizaremos el repositorio para código más popular para respaldar y llevar el control de versiones.

Cloud

Utilizaremos AWS RDS y buckets para almacenamiento.

jQuery

Aprenderás a utilizar una de las primeras librerías de Javascript.

Temas


  • Programación Orientada a Objetos

    Aprenderás el paradigma POO, para que la programación sea mucho más sencilla y los datos sean más sencillos de utilizar.

  • Diseño

    Veremos los principios de UI / UX para diseñar nuestras páginas web.

  • Herramientas

    Utilizaremos herramientas que nos ayudarán a nuestro diseño, bootstrap, animate css, font awesome, entre otras.

  • Proyecto Final

    Se hará un proyecto final para poner en práctica los conocimientos.
